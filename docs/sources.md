# Fontes de Vagas

com potencial ocorrência em São Carlos–SP.

### Casa do Trabalhador – Prefeitura de São Carlos

- URL: <https://saocarlos.sp.gov.br/files/vagas_trabalhador.pdf>
- Tipo: PDF
- Observação: Publicação periódica de vagas locais.
- Estratégia: Download + extração de texto + filtro por palavras-chave
- Prioridade: Alta
