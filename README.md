# São Carlos Tech Jobs Aggregator

Agregador de vagas de TI com foco em São Carlos–SP.
O projeto coleta oportunidades de diferentes fontes públicas (sites, PDFs, páginas de empresas e portais)
e centraliza tudo em um único repositório de dados.

Objetivo:

- Reduzir a fragmentação de vagas de TI na cidade
- Automatizar a coleta e atualização das oportunidades
- Servir como case técnico de scraping, automação e organização de dados

Status: em desenvolvimento
